# Zero Knowledge Proof for Identity Verification

## Project Overview

Zero Knowledge Proof for Identity Verification is a secure and privacy-preserving authentication system designed to verify user identity without exposing sensitive personal information.
The project uses the concept of Zero Knowledge Proof (ZKP) along with QR-based verification to allow users to prove their identity or eligibility without directly sharing confidential documents with third parties.
Instead of repeatedly sharing documents like Aadhaar cards, degree certificates, or ID proofs, the user only shares a secure QR code generated by the system. The verifier validates the QR code and checks whether the required conditions are satisfied.

## Modules

### User Module

The user can:

- Register and login securely
- Upload identity credentials such as:
  - Photograph
  - Aadhaar/ID card
  - Degree certificate
  - Other required documents
- Generate a secure QR code
- Share only the QR code instead of all personal documents

The generated QR contains encrypted proof information required for verification.

---

### Verifier Module

The verifier (merchant/company/hotel/service provider) can:

- Login securely
- Define verification requirements
- Verify user identity using:
  - QR Code Scanning
  - QR Hash Verification

The system checks whether the user satisfies the required conditions without revealing complete user data.

If conditions match, access or permission is granted. Otherwise, access is rejected.

## Working Flow
1. User uploads required documents
2. System generates secure QR code
3. User shares only QR code
4. Verifier scans QR or verifies hash
5. System validates conditions
6. Access is granted or rejected
